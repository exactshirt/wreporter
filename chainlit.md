# Wreporter

B2B 영업팀을 위한 **기업 분석 리포트 자동 생성** 시스템입니다.

## 사용 방법

1. **Chat Profile 선택** — 상단에서 `일반정보` / `재무정보` / `임원정보` 중 하나를 선택합니다.
2. **기업 검색** — 기업명을 입력하면 검색 결과가 나타납니다.
3. **기업 핀 추가** — 검색 결과에서 원하는 기업을 핀 추가합니다.
4. **조사 시작** — `/조사`를 입력하거나 `조사 시작` 버튼을 클릭합니다.
5. **후속 질문** — 조사 완료 후 추가 질문을 할 수 있습니다.

## 분석 유형

| 프로필 | 분석 내용 |
|--------|----------|
| 일반정보 | 기업개요, AX 동향, 사업 현황, 영업 인사이트 |
| 재무정보 | 재무제표, 건전성, 투자여력 평가 |
| 임원정보 | 임원 리스트, 의사결정 구조, 주요 인물 분석 |
