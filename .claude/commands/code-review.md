# 코드 리뷰 서브에이전트

방금 작성/수정된 파일을 검토해줘.

## 체크 포인트

### 1. 에러 처리
- try/except 있는가?
- DART status "013"은 None 반환 (예외 아님)으로 처리하는가?
- 네트워크 타임아웃 처리 있는가?

### 2. 로깅
- 🟢 시작 / ✅ 완료 / ❌ 에러 패턴 적용됐는가?
- 운영자가 로그만 보고 "어디서 실패했는지" 알 수 있는가?

### 3. 코드 품질
- 타입 힌트 있는가 (`def func(name: str) -> dict`)
- 함수가 한 가지 역할만 하는가
- 변수명이 이해하기 쉬운가

### 4. 테스트 가능성
- 이 코드를 pytest로 테스트할 수 있는가?
- 테스트 파일이 있는가?

### 5. Wreporter 규칙 준수
- corp_code NULL 필터 안 걸었는가?
- service_role key 사용하는가?
- Windows 호환 코드인가?

## 결과
- 문제 있으면: 수정해줘
- 문제 없으면: "✅ 리뷰 통과" 출력

검토 파일: $ARGUMENTS
