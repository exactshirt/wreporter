# DB 연결 검증 서브에이전트

Supabase 연결과 핵심 쿼리가 정상인지 확인해줘.

## 검증 항목

### 1. 연결 확인
- db/client.py의 get_client() 호출
- 연결 성공 여부

### 2. 검색 테스트
- `search_companies("삼성")` 실행
- 결과 건수 (20건 이상이면 정상)
- corp_code가 NULL인 결과도 포함되는지 확인 (포함되어야 정상)

### 3. 단건 조회 테스트
- `get_company("00126380")` 실행
- 삼성전자가 맞는지 확인 (corp_name에 "삼성전자" 포함)

### 4. 테이블 상태
- companies 테이블 총 레코드 수
- corp_code가 있는 행 수 vs NULL인 행 수
